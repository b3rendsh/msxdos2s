MSXDOS kernel entry points:
---------------------------
The following entry points (for DOS 1 and 2) are defined in the sources 
for MSXDOS.SYS 1.03 and MSXDOS 2.2 DISK Kernel.These will be considered 
the standardized kernel entry points:

Driver:
DSKIO	EQU	4010H
DSKCHG	EQU	4013H
GETDPB	EQU	4016H
CHOICE	EQU	4019H
DSKFMT	EQU	401CH
MTOFF	EQU	401FH

Disk:
C4022	EQU	4022H	Start DiskBasic
C4025	EQU	4025H	Format disk (workarea must be supplied)
C4029	EQU	4029H	Stop all disks
C402D	EQU	402DH	GETSLT - Get slotid

DOS Kernel:
