JIO server for linux
====================

Install/run
-----------
JIOServer is a one file amd64 linux app.
Make sure the file is executable and start it.

 
Notes
-----
1. Following config file is created: ~/.config/louthrax/JIOServer.conf.
2. Add user to dialout group if access to USB port is denied.
3. Tested with Fedora, Ubuntu, Mint and other distributions.
